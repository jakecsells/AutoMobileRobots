Lab 2

Github: Lab 2

The entire lab can be downloaded from our gibhub page or downloaded from this website. The solution to lab 2 is within the folder "Lab02".

We successfully completed lab 2 by finishing each mini-program for the lab.

The most complex of the programs written is the multi-tasking program. This program required a mutex to control which function has control of the robot at a certain time.

Mutex Example:

mutex control;
...
task check_sensors()
{
  while (true)
  {
    Acquire(control);

    // If left sensor pressed
...
    Release(control);
  }
}
...
// Move forward at 50% power
task move_forward()
{
  while (true)
  {
    Acquire(control);
    OnFwd(OUT_AC, 50);
    Release(control);
  }
}
